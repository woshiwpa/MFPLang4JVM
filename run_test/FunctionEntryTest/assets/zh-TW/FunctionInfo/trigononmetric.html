程式開發科學計算器幫助：三角函數

函數名 函數幫助資訊
acos

acos(1) :

acos(x)返回x的反余弦值，註意x可以為復數。
acosd

acosd(1) :

函數acosd(x)為余弦函數的反函數,註意返回值為角度數。
asin

asin(1) :

asin(x)返回x的反正弦值，註意x可以為復數。
asind

asind(1) :

函數asind(x)為正弦函數的反函數,註意返回值為角度數。
atan

atan(1) :

atan(x)返回x的反正切值，註意x可以為復數。
atand

atand(1) :

函數atand(x)為正切函數的反函數,註意返回值為角度數。
cos

cos(1) :

cos(x)返回x的餘弦值，註意x可以為復數。
cosd

cosd(1) :

函數cosd(x)返回基於角度的x的余弦值。
sin

sin(1) :

sin(x)返回x的正弦值，註意x可以為復數。
sind

sind(1) :

函數sind(x)返回基於角度x的正弦值。
tan

tan(1) :

tan(x)返回x的正切值，註意x可以為復數。
tand

tand(1) :

函數tand(x)返回基於角度x的正切值。
