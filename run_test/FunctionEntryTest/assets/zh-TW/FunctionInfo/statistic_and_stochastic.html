函數gavg(…)返回任意個數參數的幾何平均數值。

med(0...) :

函數med(…)返回任意數目參數的中位數。如果參數的個數為偶數個，返回中間v潃荌捊漸郁＃?。
min

min(0...) :

函數min(...)返回任意數目參數中的最小值。
ncr

ncr(2) :

函數程式nCr(x, y)計算有x個元素的集合S的k個元素組合的個數。註意x，y都是非負整數，x >= y。
npr

npr(2) :

函數程式nPr(x, y)計算有x個元素的集合S的k個元素置換的個數。註意x，y都是非負整數，x >= y。

rand(0) :

stdev(0...) :

函數stdev(…)返回任意個數參數的標準差，注意這些參數是一個大的集合中的採樣。

stdevp(0...) :
