MFP編程語言help和endh語句和@language標註：

在MFP語言中，Help語句是一個幫助信息塊的開始而endh語句是一個幫助信息塊的終止。

註意雖然在一個幫助信息塊中的語句不會影響函數的運行，當用戶輸入"help 函數名"或者"help 函數名(函數參數個數)"時幫助信息塊卻能提供必要的幫助信息。條件是該幫助信息塊正好位於函數聲明的上方。在以下例子中一個幫助信息塊位於函數abcd的上方：

Help

This line will be shown for any system language.

@language:

This line will be shown for default system language.

@end

    This line will also be shown for any system language.

@language:simplified_chinese

這一行將在系統語言為中文時顯示（This line will be shown when system language is simplified Chinese.）。

@end

This line is also a line for any system language.

Endh

Function abcd(x, y)

Endf

。當用戶輸入"help abcd"或者"help abcd(2)"時，如果系統語言是英語並且英語也是默認語言，該用戶將看到如下幫助信息：

This line will be shown for any system language.

This line will be shown for default system language.

    This line will also be shown for any system language.

This line is also a line for any system language.

，如果系統語言是日語但是日語不是默認語言，該用戶將看到如下幫助信息：

This line will be shown for any system language.

    This line will also be shown for any system language.

This line is also a line for any system language.

，如果系統語言是簡體中文但簡體中文不是默認語言，，該用戶將看到如下幫助信息：

This line will be shown for any system language.

    This line will also be shown for any system language.

這一行將在系統語言為中文時顯示（This line will be shown when system language is simplified Chinese.）。

This line is also a line for any system language.

，如果系統語言是簡體中文而且簡體中文是默認語言，，該用戶將看到如下幫助信息：

This line will be shown for any system language.

This line will be shown for default system language.

    This line will also be shown for any system language.

這一行將在系統語言為中文時顯示（This line will be shown when system language is simplified Chinese.）。

This line is also a line for any system language.

。

如果用戶僅僅只是想在一行的末尾加一些註釋，可以使用類似於C++的"//"，在一行中，"//"之後的內容均為註釋。"//"可以位於一行最開始。
