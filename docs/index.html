MFP language Help

MFP programming language is a scripting tool to solve sophisticated problems by taking most advantage of today's hardware capability. MFP means Mathematical language For Parallel computing. With built-in support to object oriented programming, basic mathematic calculations (i.e. matrix, complex, etc.) and parallel computation, MFP targets at scientific researchers, engineers and software developers.

The design of MFP is platform independent. However, limited by resource, at this moment MFP can only run on JVM and Android. MFP on JVM has been open-sourced. Its source codes can be downloaded from https://github.com/woshiwpa/MFPLang4JVM. MFP on Android is an App named Scientific Calculator Plus, which is available in Google Play at https://play.google.com/store/apps/details?id=com.cyzapps.AnMath. Scientific Calculator Plus includes a full set of manual files which cover both MFP language and use of this App. This webpage also links to manual files of Scientific Calculator Plus.

MFP language is still under development. It is evolving everyday. Two tasks on the radar are:

1. Implementing JIT compiler to improve its performance;

2. Interoperation between MFP and JAVA.

Besides, a few tasks are on-going and some exciting new features of MFP and/or Scientific Calculator Plus will be exposed in new future. Tony Cui, the creator, designer and developer of MFP and Scientific Calculator Plus welcomes everyone interested in this new language to join the project and help MFP grow up.
